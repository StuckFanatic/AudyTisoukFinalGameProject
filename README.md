Game Loop:

The game however is a very walk sim where you move/explore.
There is an ending in the game below will guide you through it.
There is a monster that will roam the map in the same time as you as you explore and 
react to the map. The biggest challange will be making sure you remember where each room leads.
As you explore you will encounter the monster. Once it chases you run! Eventually it will cool off your trail.
Your health will depleate with each movement so keep an eye on that.
The monster will kill you if you run into a wall.
Example: If you and the monster end up in your cell and you try to move left, well there is no left so you hit a wall and die to the monster!


------------------------------------------------------------------------------------

Items in the game: 

Knife: Will save you if you get caught and will stun the monster
Green Herb: Heals you from 50
Keys: 4 in total, Will help you unlock certain locked doors
Boat Parts: 4 parts in total that need to be brought togehtr to repair and escape the island

-------------------------------------------

Item Locations:

Cell: Knife + Herb

BlockA: Rusty Key

Cafeteria: Knife + Herb

Visiting: Herb

Matinence Room: Smooth Key + Boat Hull

Armory: Knife + Motor

Office: Red Keycard

Labortory: Fuel

Panic Room: Knife 

Yard: Herb

Tower: Gold Key + Map

Shore: Herb

----------------------------------------------------------

Ending:

Gather all the boat parts and drop them at the reciviving docks. Once all 4 parts are there you will end the game.
